WordPress Plugin

Digital Product Bulk Uploader for WooCommerce

A lightweight tool that allows WooCommerce store owners to create multiple downloadable products in seconds by uploading files and optional matching images.

Features

  • Bulk upload downloadable product files
  • Automatic product creation
  • Filename-based image matching
  • Shared price, category, and description
  • Draft or publish on import
  • Duplicate protection

How It Works

Upload your product files, optionally upload matching images with the same names, set shared options, and import. Products are created automatically using the file names.

Availability

This plugin is currently under review for the WordPress Plugin Directory.
