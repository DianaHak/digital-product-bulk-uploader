FAQ

Common questions.

Does this plugin require WooCommerce? +
Yes. WooCommerce must be installed and active on your WordPress site. The plugin creates products directly inside WooCommerce.
Can images be uploaded in any order? +
Yes. Images are matched by filename, not by upload order. So my-book.jpg matches my-book.pdf regardless of order.
What if no image matches a file? +
The product is still created — just without a featured image. You can always add one later from the product editor.
Can I set different prices per product? +
Not in the current version. A single shared price is applied to all imported products in a batch. Per-product pricing may come in a future update.
Will it overwrite existing products? +
No. If a product with the same title already exists, it is skipped. No existing products are modified or deleted.
Can I review products before publishing? +
Yes. Choose Draft status before importing and products will be saved as drafts, letting you review and edit each one before going live.
